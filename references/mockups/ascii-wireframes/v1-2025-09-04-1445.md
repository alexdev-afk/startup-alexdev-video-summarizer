## Component Hierarchy

```
VideoSummarizerApp
├── Header (title, settings access)
├── GuidedWorkflow (main interface)
│   ├── PreparationStep (input folder status)
│   ├── ProcessingStep (start button, status)
│   └── ResultsStep (browse index, access outputs)
├── ProcessingStatus (modal/separate window)
│   ├── ProgressOverview (batch progress)
│   ├── CurrentVideoStatus (detailed current item)
│   ├── RecentCompletions (success list)
│   └── ControlButtons (pause/stop/log)
└── VideoIndexBrowser (separate interface)
    ├── SearchHeader (search, filters)
    ├── VideoTable (sortable list with metadata)
    ├── StatusLegend (icon explanations)
    └── VideoPreview (selected item details)
```

---

**Design Notes:**
- Monospace-aligned ASCII using box drawing characters
- Clear step-by-step workflow for non-technical users  
- Visual status indicators (✅, ⏳, ⚠️, 🔄)
- Duplicate detection flagged with icons in index
- Processing confidence scores visible
- Simple navigation between main states
