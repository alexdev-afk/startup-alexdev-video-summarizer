## **User Flow Overview**

### **Primary Happy Path**
1. **Launch** → Show available videos and configuration
2. **Start Processing** → Real-time progress with scene-by-scene feedback  
3. **Monitor Progress** → 7-tool pipeline status with detailed activity log
4. **View Results** → Summary with direct access to knowledge base files

### **Error Recovery Flow**
1. **Error Detection** → Circuit breaker stops after 3 failures
2. **Error Display** → Clear categorization of failures vs. successes
3. **Recovery Options** → Retry individually, skip failed, or investigate logs
4. **Continuation** → Resume processing or exit with partial results
